Colors ERD

ERD Description

Table PrimaryColor

  • This table shows primary color names and identifying numbers. It is used to sort primary colors.
  • The Id attribute gives each color a uniqe number for identification.
  • Under PrimaryColorName attribute are color names.
  • This table itself does not have a foreign key, but the Id is used as a foreign key in the Shades table.

Table Shades

  • This table is used to sort color shades and see what primary colors they connect with using the PrimaryColorId ttribute.
  • The Id attribute gives each shade a uniqe identification number, so shades dont mix in the table.
  • PrimaryColorId attribute is used to connect primary color to a shade.
  • ColorName attribute is used to know what color shade there is.
  • This table uses primary key Id from PrimaryColor table as a foreign key. Shades can have one primary color but colors can have many shades.
